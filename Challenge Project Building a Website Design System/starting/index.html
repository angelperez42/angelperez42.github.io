Colors

KNTY Blue

#0070C0

This color conveys modernity and fun, should be used to contrast white or braige backgrounds.

Drop Braige

#D6D3CC

This color is relaxing and pleasing, should be used as a background but in moderation.

Sus Blay

#92A9B9

This color is a suspicious gray-blue, should be used for any secondary features in the site.
